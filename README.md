# Installation

## Just testing

```sh
npx degit lamualfa/svelte-windicss-preprocess-testing-v3 folder-name
cd folder-name
pnpm i
pnpm dev
```

## Clone

```sh
git clone https://github.com/lamualfa/svelte-windicss-preprocess-testing-v3.git
cd svelte-windicss-preprocess-testing-v3
pnpm i
pnpm dev
```
